# Face

A CLI for emojis

## Installation

Clone the repository

	cd /opt
    git clone https://github.com/daviscodesbugs/face.git

Make `face` executable

    cd face/
	chmod +x ./face

Create symlink

    sudo ln -sf /opt/face/face /usr/local/bin/face

## Usage

See available faces

    face help
